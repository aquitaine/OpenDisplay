# NativeControlProvider

**macOS target.** Native Apple/built-in brightness and audio control, plus a software
overlay/gamma dimmer for ranges below the hardware minimum (PRD CTL-001/003, combined curve
CTL-004). Surfaces the active provider and fallback so the UI can explain which route is in use.

Implements: `ControlProvider`. Milestone: **M1/M2**.
