# CaptureProvider

**macOS target.** ScreenCaptureKit-backed picture-in-picture, display zoom, and screenshots
(PRD VIR-004..007). Requests Screen Recording permission only when a capture feature is
started; denial leaves topology/controls/scenes/lifecycle fully functional. Honors system
exclusions and stops sessions on lock/logout.

Implements: `CaptureProvider`-style protocol. Milestone: **Core 1.x (M4)**.
